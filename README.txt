CreepyCoder - stalking programmers through commit messages
----------------------------------------------------------
`
This is a set of PHP5 scripts for collecting information about programmers from their commit 
messages and writing the data in other useful forms.

Configuration is done with a XML file that configures where to collect data from and where 
to write output to.

To run the program:

php src/CreepyCoder.php -c config.xml
